# Arena Shooter — 3D FPS

A browser-based 3D first-person shooter in an arena against one AI opponent.

## How to play

1. Open `index.html` in a modern browser (Chrome, Firefox, Safari, Edge).
2. **Click** the game area to lock the mouse pointer for looking around.
3. **Move:** Arrow keys or WASD.
4. **Look:** Move the mouse.
5. **Shoot:** Space bar.

Reduce the enemy’s health to 0 to win. Avoid their shots and use the pillars for cover.

## Run locally (optional)

From the project folder:

```bash
npx serve .
```

Then open the URL shown (e.g. http://localhost:3000).

## Tech

- **Three.js** (r128) for 3D rendering
- No build step; plain HTML + JS
